    A Brief Review of AI Coding Assistants With Free of Charge#

    All-In-One#

    • Codeium: Supports a lot of languages, both chat and auto-completion
    • Codium: Limited to JavaScript, TypeScript, and Python.
    • CodeGeeX: Free, FOSS, supports 10+ languages
    • Embold: Static Code Analysis, Automatic documentation, I can't make it work.
    • Cody by SourceGraph: is not available on IntelliJ, Scheduled for June 28.
    • refact: support both IJ, VC. seems mainly focusing on refactoring. seems not promising.

    Unit Testing#

    • GoCodeo: is not available on IntelliJ, limited to JavaScript/TypeScript and Python

    Teaching#

    • ExplainDev: is not available on IntelliJ, it suggests follow-up questions, and I didn't see any difference from other chat services without that. I didn't tried.

    I'm still investigating. content will be updated.
